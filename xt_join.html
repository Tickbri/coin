XT 거래소 (XT.com) 가입 및 KYC 인증

가입 절차 (웹사이트 기준)

  • 1. 가입 페이지 접속:

    XT.com 공식 홈페이지 또는 '매매법 탐구생활'의 제휴 링크를 통해 접속 후, 우측 상단의 "회원가입(Sign Up)" 버튼을 클릭합니다.

  • 2. 가입 방법 선택:

    이메일 또는 휴대폰 번호 중 편리한 가입 방법을 선택합니다.

  • 3. 정보 입력:

    선택한 방법에 따라 이메일 주소 또는 휴대폰 번호를 입력하고, 사용할 비밀번호를 설정합니다. (추천인 코드가 있다면 입력합니다.)

  • 4. 인증 코드 입력:

    입력한 이메일 또는 휴대폰으로 6자리 인증 코드가 발송됩니다. 인증 페이지에서 해당 코드를 입력하면 계정 생성이 완료됩니다.

KYC 실명 인증 방법

XT.com의 KYC 인증은 '초기 인증'과 '고급 인증'으로 나뉘며, 등급에 따라 거래 한도가 달라집니다. (XT 지원센터 정보 기준)

  • 1. 인증 센터 이동 (웹 기준):

    로그인 후, 우측 상단의 [개인센터] 아이콘을 클릭한 뒤 [실명인증] 메뉴로 이동합니다.

  • 2. 인증 방법 선택:

    PC 웹캠이 있는 경우 '사진 인증'을, 없는 경우 '앱 인증'을 선택합니다. '앱 인증' 선택 시, XT 앱으로 QR 코드를 스캔하여 모바일에서 이어서 진행합니다.

  • 3. 신분증 업로드:

    앱 또는 웹의 안내에 따라 신분증(여권, 운전면허증 등)을 촬영하여 업로드합니다.

  • 4. 얼굴 인식 (셀카):

    안내에 따라 본인의 얼굴을 촬영(셀카)하거나 실시간 안면 인식을 수행합니다.

  • 5. 제출 및 대기:

    모든 정보 제출이 완료되면, 검토가 진행됩니다. 승인 완료 시 이메일 등으로 알림이 전송됩니다.
