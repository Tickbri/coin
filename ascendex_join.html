어센덱스 (AscendEX) 가입 및 KYC 인증

어센덱스 가입하러 가기 (증정금 지급, 수수료 할인)

제공된 링크를 클릭하면 수수료 혜택이 자동 적용됩니다. 아래 순서대로 천천히 따라와 주세요.

1. 어센덱스(AscendEX) 가입 절차

1.1 가입 페이지 이동

'가입하러 가기' 버튼을 클릭하여 어센덱스 가입 페이지로 이동합니다.

1.2 가입 방법 선택

이메일 또는 휴대폰 번호 중 편리한 방법을 선택합니다.

1.3 정보 입력

선택한 정보(이메일/휴대폰)와 사용할 비밀번호를 입력합니다.

1.4 인증 코드 입력

[인증 코드 받기]를 클릭하여 받은 인증 코드를 입력합니다.

1.5 초대 코드 확인

추천인 코드(URESPHHZW)가 자동으로 입력되어 있는지 확인합니다.

1.6 가입 완료

이용약관 동의 후, '가입' 버튼을 클릭하면 계정 생성이 완료됩니다.

2. KYC 신원 인증 방법

가입 후 입금, 거래, 출금을 위해 Lv.1 인증을 권장합니다.

2.1 인증 메뉴 이동

로그인 후, [프로필 아이콘] - [본인 인증] 메뉴로 이동합니다.

2.2 인증 시작

Lv.1 [인증하기] 버튼을 클릭합니다.

2.3 정보 선택

국가(South Korea)와 신분증 유형(신분증, 운전면허증, 여권)을 선택합니다.

2.4 신분증 업로드

안내에 따라 신분증의 앞면과 뒷면을 촬영하여 업로드합니다.

2.5 얼굴 인증

본인 확인을 위한 '셀카' 또는 얼굴 인식 절차를 진행합니다.

2.6 제출 완료

정보 제출 후, 심사가 완료되면(보통 몇 분 소요) 인증이 완료됩니다.

3. 구글 OTP (2FA) 보안 설정

계정 보안을 위해 KYC 완료 후 구글 OTP(2FA) 설정을 강력히 권장합니다.

3.1 보안 설정 이동

[프로필 아이콘] - [보안 설정] 메뉴로 이동합니다.

3.2 OTP 설정 시작

[Google OTP] 항목에서 [설정]을 클릭합니다.

3.3 앱 연동

구글 OTP 앱을 설치하고, 화면의 QR코드 또는 설정 키를 등록합니다.

3.4 연동 완료

앱에 표시되는 6자리 OTP 번호를 입력하여 연동을 완료합니다.
