어센덱스 (AscendEX) 가입 및 KYC 인증 가이드

어센덱스 가입하러 가기

아래 단계를 순서대로 따라 하면, 가입부터 인증까지 5분이면 완료됩니다.


1. 어센덱스(AscendEX) 회원가입


① 가입 페이지 이동

상단의 [가입하러 가기] 버튼을 클릭하면 공식 AscendEX 가입 페이지로 이동합니다.

② 가입 방식 선택

이메일 또는 휴대폰 번호 중 하나를 선택해 가입할 수 있습니다.

③ 정보 입력

선택한 정보와 사용할 비밀번호를 입력한 뒤, 인증 코드를 받아 입력합니다.

④ 초대 코드 확인

추천 코드 URESPHHZW가 자동 입력되어 있는지 확인합니다.

⑤ 가입 완료

약관 동의 후 [가입] 버튼을 누르면 계정 생성이 완료됩니다.


2. KYC 신원 인증

입금 및 출금을 위해 Lv.1 인증이 필요합니다. 절차는 간단하며 보통 몇 분 내로 승인됩니다.


① 본인 인증 메뉴 이동

로그인 후, [프로필 아이콘] → [본인 인증] 메뉴를 클릭합니다.

② 인증 정보 입력

국가(대한민국)와 신분증 종류(주민등록증/운전면허증/여권)를 선택합니다.

③ 신분증 업로드

안내에 따라 신분증 앞/뒷면을 업로드하고, 얼굴 인증을 진행합니다.


3. 구글 OTP (2FA) 설정

해킹을 방지하기 위해 반드시 구글 OTP를 설정하세요. 로그인, 출금 시 추가 인증이 가능합니다.


① [보안 설정] 메뉴 이동

프로필 아이콘 → [보안 설정]을 클릭합니다.

② Google OTP 등록

QR코드를 스캔하거나 설정 키를 입력하여 앱에 등록합니다.

③ 연동 완료

앱에서 생성된 6자리 코드를 입력하면 OTP 설정이 완료됩니다.
