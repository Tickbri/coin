어센덱스 (AscendEX) 가입 및 KYC 인증

어센덱스 가입하러 가기 (증정금 지급, 수수료 할인)

1. 어센덱스(AscendEX) 가입 절차

제공된 링크를 클릭하여 가입 페이지로 이동하면 수수료 혜택이 자동 적용됩니다.

  1. '가입하러 가기' 버튼을 클릭하여 어센덱스 가입 페이지로 이동합니다.
  2. 가입 방법으로 **이메일** 또는 **휴대폰 번호**를 선택합니다.
  3. 선택한 정보(이메일/휴대폰)와 사용할 비밀번호를 입력합니다.
  4. [인증 코드 받기]를 클릭하여 이메일 또는 문자로 받은 인증 코드를 입력합니다.
  5. 초대 코드(URESPHHZW)가 자동으로 입력되어 있는지 확인합니다.
  6. 이용약관 동의 후, '가입' 버튼을 클릭하면 계정 생성이 완료됩니다.

2. KYC 신원 인증 방법

가입 후 입금 및 거래, 출금을 위해서는 KYC 신원 인증이 필요합니다. (Lv.1 인증 권장)

  1. 로그인 후, [프로필 아이콘] - [본인 인증] 또는 [Identity Verification] 메뉴로 이동합니다.
  2. Lv.1 [인증하기] 버튼을 클릭합니다.
  3. 국가(South Korea)와 신분증 유형(신분증, 운전면허증, 여권 중 선택)을 선택합니다.
  4. 안내에 따라 신분증의 앞면과 뒷면을 촬영하여 업로드합니다.
  5. 본인 확인을 위한 '셀카' 또는 얼굴 인식 절차를 진행합니다.
  6. 정보 제출 후, 심사가 완료되면(보통 몇 분~몇 시간 소요) KYC 인증이 완료됩니다.

3. 구글 OTP (2FA) 보안 설정

계정 보안을 위해 KYC 완료 후 구글 OTP(2FA) 설정을 권장합니다.

  1. [프로필 아이콘] - [보안 설정] 또는 [Security] 메뉴로 이동합니다.
  2. [Google OTP] 항목에서 [설정]을 클릭합니다.
  3. 안내에 따라 구글 OTP 앱을 설치하고, 어센덱스에서 제공하는 QR코드 또는 설정 키를 등록합니다.
  4. 표시되는 6자리 OTP 번호를 입력하여 연동을 완료합니다.
